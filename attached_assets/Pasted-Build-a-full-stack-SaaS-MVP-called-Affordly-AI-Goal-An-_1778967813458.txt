Build a full-stack SaaS MVP called "Affordly AI".

Goal:
An AI-powered financial decision simulator where users enter their income, expenses, savings, debts, and a future decision like "Can I afford a £12,000 car?" The app calculates affordability, projects future cashflow, and generates an AI explanation.

Tech requirements:
- Use Next.js with React
- Use Tailwind CSS
- Use TypeScript
- Use PostgreSQL database
- Use Prisma ORM
- Use Stripe subscription structure placeholder
- Use OpenAI API placeholder for AI insights
- Make the UI modern, clean, mobile-friendly, and SaaS-style

Core pages:
1. Landing page
2. Dashboard
3. New simulation page
4. Results page
5. Pricing page

Main MVP features:
- User can enter:
  - monthly income
  - monthly rent/housing
  - food expenses
  - transport expenses
  - subscriptions
  - other expenses
  - savings balance
  - debt balance
  - planned decision cost
  - optional monthly extra cost
  - scenario name

Simulation logic:
- Calculate total monthly expenses
- Calculate monthly cashflow
- Calculate savings rate
- Calculate how many months savings will last if cashflow is negative
- Calculate affordability score from 0 to 100
- Classify result as:
  - Safe
  - Moderate
  - Risky
  - Dangerous

AI insight:
Create a function called generateAIInsight(simulationData) that currently returns a realistic placeholder explanation.
Add comments showing where to connect OpenAI API later.

UI:
- Dashboard cards for income, expenses, savings, affordability score
- Charts using Recharts
- Result screen should show:
  - affordability score
  - risk level
  - projected 12-month savings
  - AI explanation
  - simple recommendations

Important:
- Do not give financial advice
- Add disclaimer: "This tool provides educational simulations only and is not financial advice."
- Make the app look production-ready
- Include dummy demo data
- Make sure the app runs without needing real Stripe or OpenAI keys
